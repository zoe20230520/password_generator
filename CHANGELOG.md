## [v2.0.2] - 2026-02-02

### 新增

#### 收藏网站功能增强
- 添加收藏类型选择（链接、图片、文章）
- 添加网址URL输入框，支持默认示例
- 添加封面图片URL输入框，支持实时预览
- 实现URL自动识别功能（提取标题和描述）
- 根据收藏类型动态显示表单字段
- 添加时间标注功能（显示相对时间，如"大约2小时前"）
- 添加分类导航（所有收藏、链接、图片、文章、设置）
- 添加统计卡片显示各类型收藏数量

### 优化

#### 用户体验
- 优化添加收藏表单布局和交互
- 添加表单描述文字，提升用户体验
- 更新按钮文案（保存 -> 添加）
- 优化图片预览功能

## [v2.0.1] - 2026-02-02

### 重大变更

#### 功能重构
- 将"智能剪贴板"功能重构为"收藏网站"应用
- 重命名前端文件：clipboard.html -> favorites.html, clipboard.js -> favorites.js
- 重命名后端数据模型：ClipboardItem -> FavoriteItem, ClipboardUsage -> FavoriteUsage
- 修改API路由：/api/clipboard -> /api/favorites

#### 功能定位调整
- 从文本剪贴板管理转向网站链接、图片、文章收藏管理
- 保留原有的分类、标签、使用统计等核心功能
- 优化UI界面，更新图标和文案

### 新增

#### 收藏网站功能
- 收藏网页链接管理
- 收藏图片管理
- 收藏文章管理
- 支持AI助手自动总结网站内容和用途（待实现）

### 优化

#### 用户体验
- 更新导航栏和入口文案
- 优化页面标题和描述
- 统一使用星形图标表示收藏功能

## [v1.0.0] - 2026-01-30

### 新增

#### 核心功能
- 用户注册和登录系统
- JWT Token 身份验证
- 密码记录的增删改查
- 密码显示/隐藏切换
- 一键复制密码到剪贴板

#### 密码生成器
- 自定义密码长度（6-32位）
- 支持字母、数字、符号组合
- 实时密码强度评估（5个等级：极弱/弱/中等/强/极强）
- 一键复制功能

#### 分类与管理
- 密码记录分类管理
- 搜索功能（支持网站名称、用户名、备注）
- 分页显示（20条/页）
- 分类筛选功能

#### 用户界面
- 现代化玻璃态设计风格
- 响应式布局，支持移动端
- 流畅的交互体验
- 优化的登录/注册页面

#### 数据安全
- 密码加密存储（Werkzeug）
- 用户数据完全隔离
- 本地 SQLite 数据库
- 基于 Token 的会话管理

#### 文档
- 完整的用户使用指南（USER_GUIDE.md）
- 详细的故障排除指南（TROUBLESHOOTING.md）
- 客户交付清单（DELIVERY_CHECKLIST.md）
- 完善的 README 文档

### 技术栈
- **后端**: Python 3.7+, Flask 3.0.0, SQLAlchemy 3.1.1
- **数据库**: SQLite 3
- **认证**: JWT (PyJWT 2.8.0)
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **UI 框架**: Bootstrap 5, Font Awesome 6
- **图片处理**: Pillow 10.1.0

### 浏览器支持
- Chrome 60+
- Firefox 55+
- Edge 79+
- Safari 11+

### 安全特性
- PBKDF2 密码哈希
- JWT Token 认证
- Token 有效期 7 天
- 数据库表权限隔离
- 输入验证和错误处理

## 版本说明

### 版本号格式
格式为：主版本号.次版本号.修订号 (MAJOR.MINOR.PATCH)

- **主版本号 (MAJOR)**：不兼容的 API 修改
- **次版本号 (MINOR)**：向下兼容的功能性新增
- **修订号 (PATCH)**：向下兼容的问题修正
