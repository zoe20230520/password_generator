# 变更日志

本文档记录密码管理系统的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [v1.0.1] - 2026-01-30

### 新增

#### 图片功能
- 图片预览优化为150x150px小图显示
- 点击图片可放大查看完整尺寸
- 新增图片放大模态框组件
- 图片加载完成和错误处理机制

### 修复

#### 图片显示
- 修复图片无法显示的问题
- 移除图片访问的认证要求，优化加载性能

---

## [v1.0.0] - 2026-01-30

### 新增

#### 核心功能
- 用户注册和登录系统
- JWT Token 身份验证
- 密码记录的增删改查
- 密码显示/隐藏切换
- 一键复制密码到剪贴板

#### 密码生成器
- 自定义密码长度（6-32位）
- 支持字母、数字、符号组合
- 实时密码强度评估（5个等级：极弱/弱/中等/强/极强）
- 一键复制功能

#### 分类与管理
- 密码记录分类管理
- 搜索功能（支持网站名称、用户名、备注）
- 分页显示（20条/页）
- 分类筛选功能

#### 用户界面
- 现代化玻璃态设计风格
- 响应式布局，支持移动端
- 流畅的交互体验
- 优化的登录/注册页面

#### 数据安全
- 密码加密存储（Werkzeug）
- 用户数据完全隔离
- 本地 SQLite 数据库
- 基于 Token 的会话管理

#### 文档
- 完整的用户使用指南（USER_GUIDE.md）
- 详细的故障排除指南（TROUBLESHOOTING.md）
- 客户交付清单（DELIVERY_CHECKLIST.md）
- 完善的 README 文档

### 技术栈
- **后端**: Python 3.7+, Flask 3.0.0, SQLAlchemy 3.1.1
- **数据库**: SQLite 3
- **认证**: JWT (PyJWT 2.8.0)
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **UI 框架**: Bootstrap 5, Font Awesome 6
- **图片处理**: Pillow 10.1.0

### 浏览器支持
- Chrome 60+
- Firefox 55+
- Edge 79+
- Safari 11+

### 安全特性
- PBKDF2 密码哈希
- JWT Token 认证
- Token 有效期 7 天
- 数据库表权限隔离
- 输入验证和错误处理

## 版本说明

### 版本号格式
格式为：主版本号.次版本号.修订号 (MAJOR.MINOR.PATCH)

- **主版本号 (MAJOR)**：不兼容的 API 修改
- **次版本号 (MINOR)**：向下兼容的功能性新增
- **修订号 (PATCH)**：向下兼容的问题修正
