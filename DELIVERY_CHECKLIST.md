## 客户可能遇到的问题

### 问题 1：启动失败

**原因**：
- Python 版本过低
- 依赖安装失败

**解决方案**：
- 检查 Python 版本（需要 3.7+）
- 手动运行：`pip install -r backend\requirements.txt`

### 问题 2：端口被占用

**原因**：5000 端口已被其他程序使用

**解决方案**：
修改 `backend\app.py` 最后一行，将端口改为其他值：
```python
app.run(debug=True, host='0.0.0.0', port=5001)
```

### 问题 3：忘记密码

**说明**：系统不支持密码找回功能

**建议**：
- 告诉客户牢记主密码
- 可以删除 `database.db` 文件重新注册

### 问题 4：数据备份

**建议**：
定期备份 `database.db` 文件
